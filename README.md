# Job Hunt Pipeline

A structured, non-negotiable, five-prompt AI pipeline that takes your resume from raw to ATS-optimized, hiring-manager-ready, and matched to 100+ verified jobs across three search tracks.

Built with **Vite** and **React**, featuring an interactive Three.js background, modern design cues, and instant downloads for all pipeline assets.

## The System

1. **Profile Analysis**: Flag skill gaps, identify competitive roles, and extract your strongest selling points.
2. **ATS Scoring**: Diagnose across 8 ATS parsing dimensions with zero rewrites to get an honest baseline score.
3. **Resume Rewrite**: Comprehensive ATS-safe plain-text rewrite based strictly on the ATS gap analysis.
4. **Hiring Manager Review**: Four distinct human calibration lenses to score domains, seniority, and consistency. No version proceeds unless rated "Ready".
5. **Job Search**: Automate candidate exclusions, spin up primary/adjacent job matching, and fetch 100+ vetted openings natively.

## Getting Started

Since this project has been fully migrated to a React codebase:

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Start Development Server**:
   ```bash
   npm run dev
   ```
3. **Build for Production**:
   ```bash
   npm run build
   ```

## Assets

All required pipeline prompts and AI skill definitions (`.txt` & `.md`) reside securely in the `public/` folder, allowing robust direct downloads via the frontend user interface.
